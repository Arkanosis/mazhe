*****************************************************************************
DEPENDENCIES
*****************************************************************************

You need 
1. a cool LaTeX distribution;
2. the package exocorr.sty (see Gitorious);
3. the python's module phystricks (see Gitorious)

exocorr.sty
===========
C'est le paquet qui sert à gérer les exercices et les correction. Il est à mettre dans le répertoire 
~/texmf/tex/latex/exocorr
Deux façons d'obtenir le fichier. 
1. Soit le télécharger à http://www.gitorious.org/exocorr
2. Soit par git :
> git clone git://gitorious.org/exocorr/exocorr.git
dans le répertoire ~/texmf/tex/latex

*****************************************************************************
COMPILATION
*****************************************************************************

The figures are generated by an external script in python using the module phystricks (see gitorious), or by sending an email to the author <moky.math@gmail.com> asking for pdf's or pstricks files.

pdflatex
========
1. Generate the figures
	./figure_modelisation.py --all --pdf
2. Compile
	pdflatex GeomAnal.tex

Chain dvi->ps->pdf
==================
1. Generate the figures
	./figure_GeomAnal --all
2. Compile
	latex --src-specials GeomAnal.tex
	dvips -t a4 GeomAnal.dvi
	ps2pdf GeomAnal.ps

Index, bibliography and list of notations
=========================================
	bibtex GeomAnal
	makeindex GeomAnal
	makeindex -s nomencl.ist -o GeomAnal.nls GeomAnal.nlo
