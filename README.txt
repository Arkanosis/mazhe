*****************************************************************************
DEPENDENCIES
*****************************************************************************

You need 
1. a cool LaTeX distribution;
2. the package exocorr.sty (see Gitorious);
3. the python's module phystricks (see Gitorious)

exocorr.sty
===========
C'est le paquet qui sert à gérer les exercices et les correction. Il est à mettre dans le répertoire 
~/texmf/tex/latex/exocorr
Deux façons d'obtenir le fichier. 
1. Soit le télécharger à http://www.gitorious.org/exocorr
2. Soit par git :
> git clone git://gitorious.org/exocorr/exocorr.git
dans le répertoire ~/texmf/tex/latex

*****************************************************************************
COMPILATION
*****************************************************************************

The figures are generated by an external script in python using the module phystricks (see gitorious), or by sending an email to the author <moky.math@gmail.com> asking for pdf's or pstricks files.

pdflatex
========
1. Generate the figures
	./figure_GeomAnal.py --all --pdf
2. Compile
	pdflatex GeomAnal.tex

Chain dvi->ps->pdf
==================
1. Generate the figures
	./figure_GeomAnal --all
2. Compile
	latex --src-specials GeomAnal.tex
	dvips -t a4 GeomAnal.dvi
	ps2pdf GeomAnal.ps

Index, bibliography and list of notations
=========================================
	bibtex GeomAnal
	makeindex GeomAnal
	makeindex -s nomencl.ist -o GeomAnal.nls GeomAnal.nlo

*****************************************************************************
THE ctu ISSUE
*****************************************************************************

In order to make an "non-CTU compatible" file, some changes are to be done.
To produce a "CTU-compatible" file, the changes are to be done the other way.

At the very beginning of e_GeomAnal.tex, there is the line

\newcounter{ThisIsCTU}
\setcounter{ThisIsCTU}{0}

If you want a CTU-compatible document, set ThisIsCTU to 1
If you want a non CTU-compatible document, set ThisIsCTU to 0

The detail of what is done is in the next two paragraphs.

In GeomAnal.tex
===============
Uncomment \input{garde} and comment the three others :

\input{garde}
%\frontmatter
%\input{garde_officielle}
%\pagestyle{ctu}

Comment the lines about \mainmatter and the page counter fixing hack

%\newcounter{bidon}
%\setcounter{bidon}{\value{page}}
%\mainmatter
%\stepcounter{bidon}
%\setcounter{page}{\value{bidon}}


Uncomment the list of notations and the index
	\addcontentsline{toc}{chapter}{Liste des notations}
	\printnomenclature
	\printindex

In e_GeomAnal.tex
=================
Comment livre3
%\usepackage[imprime,perse]{livre3}

Comment microtype and fancyhdr and uncomment geometry :
%\usepackage{microtype}		% Pour la page de garde CTU
%\usepackage{fancyhdr}		% Pour la page de garde CTU
\usepackage[margin=2cm]{geometry} 

Uncomment some definitions
\newtheorem{definition}[numtho]{Définition}
\newtheorem{proposition}[numtho]{Proposition}
\newtheorem{theoreme}[numtho]{Théoreme}

Uncomment makeidx
\usepackage{makeidx}
